## Step 4 — Fix `main.py` before building

Before we build, we need to patch the DB path issue. Open `main.py` in PyCharm and find these two lines near the top:

```python
DB_DIR  = os.path.join(os.path.dirname(os.path.abspath(__file__)), "tickers")
DB_OUTPUT  = os.path.join(os.path.dirname(os.path.abspath(__file__)), "output")
```

Replace them with:

```python
_BASE = os.path.dirname(sys.executable) if getattr(sys, 'frozen', False) else os.path.dirname(os.path.abspath(__file__))
DB_DIR    = os.path.join(_BASE, "tickers")
DB_OUTPUT = os.path.join(_BASE, "output")
```

Save the file.
